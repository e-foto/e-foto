# e-foto
The goal of Project E-foto is to provide the full implementation of 
a digital photogrammetric workstation as free software under a GNU GPL license.

## Install
### deb package for linux
1. Download the latest e-foto deb package from our [website](http://www.efoto.eng.uerj.br/download/latest-version) 
2. Double click on the downloaded file or, at a terminal, type: 
   > sudo dpkg -i e-foto-<version-number>.deb
3. Follow the instructions on screen
